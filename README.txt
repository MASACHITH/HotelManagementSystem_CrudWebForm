# Hotel Management System - CRUD Web Forms
This is a Hotel Management System built using:
- ASP.NET Web Forms (Frontend)
- ASP.NET Web API (Backend)
- Dapper for Database Access
- MySQL / SQL Server for Database

## Features:
- Manage Room Reservations
- Folio Transfers
- Guest Management
- REST API Integration

## Installation:
1. Clone the repository
2. Open in Visual Studio
3. Restore NuGet packages
4. Run the Web API (`backend/`) and Web Forms (`frontend/`)
